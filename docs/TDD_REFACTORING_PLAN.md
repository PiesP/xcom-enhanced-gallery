3. **YAGNI 원칙** (Phase 107-108)

   - 작업 가치 평가: 작업량 대비 실질적 효과를 먼저 평가

4. **과도한 최적화 경계**

   - 타입 단언 27개 중 15개는 시스템 설계상 필수
